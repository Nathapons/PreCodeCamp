Tag name Attribute คำอธิบาย
h Nested สร้างหัวเรื่องขึ้นขนาดจะลดลงตามจำนวนตัวเลขมากกขึ้น (h1-h6)
p Nested แสดงรายละเอียดข้อความทั่วไป
div Nested ทำหน้าที่เป็น Container ในการเก็บข้อมูล
img Empty ทำหน้าที่แทรกรูปภาพจากลิงค์เว็บไซต์หรือจากรูปภาพไฟล์พาทในคอมพิวเตอร์
a Nested เป็นคำสั่งในการย้ายจากหน้าต่างที่นำเสนอไปยังลิงค์ต่างๆ
ul Nested เรียงรูปแบบโดยมี ● อยู่ข้างหน้า
ol Nested เรียงรูปแบบโดยเรียงตามตัวเลข
table Nested/td> สร้างตาราง
form Nested เป็นรูปแบบฟอร์มสำหรับการ Input
button Nested สร้างปุ่มในการกด
