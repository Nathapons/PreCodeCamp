คำสั่ง ความหมาย/ความต่าง
hr & br hr=สร้างเส้นในแนวนอนแต่ br จะทำหน้าที่ขึ้นบรรทัดใหม่
ul & ol ทำหน้าที่ลิสทั้งคู่แต่จะต่างกันตัวอักษรกำกับ list: ul ใช้ ●, ol ใช้ตัวเลข
select & checkbox & radiobutton

select เป็นแท็กในการระบุใน document, checkbox จะเป็นตัวเลือกช้อยในรูปแบบติ๊กถูกสามารถเลือกได้หลายๆอัน, radiobutton จะเป็นตัวเลือกช้อยในรูปแบบ ● สามารถเลือกได้อันเดียวเท่านั้น

div & span div และ span จะทำหน้าที่เหมือน container ในการเก็บข้อมูลแต่จะแตกต่างข้ออความถัดไป div: ข้อความจะอยู่บรรถัดไป, span ข้อความจะอยู่ข้างๆ
script & style script และ style เปรียบเสมือนกล่องใหญ่ในการเก็บคำสั่ง แต่มันจะแตกต่างกันรูปแบบคำสั่งในกล่อง script จะเก็บคำสั่ง Javascript ส่วน style จะเก็บคำสั่ง css เอาไว้
