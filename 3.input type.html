type ใน attribute: input คุณลักษณะ
button ปุ่มกด
checkbox ช่องสำหรับเช็คถูก สามารถเลือก checkbox หลายๆอันได้
color เลือก shade สีในรูปแบบ RGB, HSL, HEX
date เลือก date หลังจากเลือกจะระบุเป็น mm/dd/yyyy
datetime-local เลือก datetime-local หลังจากเลือกจะระบุเป็น mm/dd/yyyy:เวลา
e-mail เป็นช่องกรอกสำหรับ e-mail
file จะมีปุ่มาสำหรับเลือกไฟล์
hidden เป็นพื้นที่ว่างเปล่า
Image รูปภาพสำหรับคลิก
month เลือก datetime-local หลังจากเลือกจะระบุเป็น month
number เป็นช่องสำหรับปรับค่าขึ้นลงของตัวเลข
password ช่องกรอกและแสดงผล ● เมื่อกรอกข้อมูลลงไป
radio ช่องสำหรับเช็ค ● และเปลี่ยนจุดสีดำใหม่เหมือน radio ที่อีกตำแหน่ง
range จะมีเส้นยาวไว้สำหรับ drag ไปมา
reset จะเป็นปุ่มสำหรับการ reset input type ทั้งหมด
search จะเป็นช่องสำหรับกรอกคล้าย text แต่จะมีปุ่ม x หากต้องการลบข้อมูลในช่อง search ออก
submit ปุ่มกด submit
tel เป็นช่องกรอกสำหรับเบอร์โทร
text เป็นช่องกรอกสำหรับข้อความ
time เลือก time หลังจากเลือกจะระบุเป็น เวลา pm/am
url เป็นช่องกรอกวำหรับ url
week เลือก week หลังจากเลือกจะระบุเป็น Week จำนวนสัปดาห์ใน 1 ปี, ปีที่เลือก
